# README

this is a template for ADA reports. you use it by doing 

```
quarto render report.ipynb
```
if your report is called `report.ipynb`. As long as the report is in the same directory as the `_quarto.yml` file, it should produce a customized report. 

You can use it by cloning the repository. However, if you already have a git repository in the same directory you may get into trouble. Always clone into a separate sub directory using:

```
git clone github.com/ADAorg/report_template report/
```
where report/ is the directory where you will have your report.
